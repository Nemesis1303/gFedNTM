src.models.base.pytorchavitm.utils.data_preparation module

src.models.base.pytorchavitm.utils.data_preparation.prepare_dataset(corpus, val_size=0.25)[source]

It prepares the training data in the format that is asked as input in AVITM.

Parameters

  • corpus (List[str]) – List of documents to be used for training of the model

  • val_size (float (default=0.25)) – Percentage of the documents to be used for validation

Returns

  • train_data (BOWDataset) – Training dataset in the required format for AVITM

  • val_data (BOWDataset) – Validation dataset in the required format for AVITM

  • input_size (int) – Size of the input dimensions of the AVITM model to be trained

  • id2token (tuple) – Mappings with the content of each training dataset’s document-term matrix.
